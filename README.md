## O projekcie

- Silnik: Unity
- Platforma: VR

## Jak uruchomić

1. Otwórz projekt w Unity.
2. Wczytaj scenę startową.
3. Uruchom grę w trybie Play.

## Uwagi

Jeśli chcesz, możesz zmienić nazwy plików obrazków na własne i podmienić je w sekcji `Zdjęcia`.
